# 더라운드 (The Round) 웹사이트 프로젝트 지침

이 파일은 **Gemini CLI**가 이 프로젝트의 문맥을 이해하고 일관성 있게 작업을 이어가기 위한 가이드입니다. 다음 세션 시작 시 이 파일을 먼저 읽고 작업을 진행하십시오.

## 1. 프로젝트 개요
- **목적**: 비영리 민간단체 '더라운드'의 공식 웹사이트 구축
- **기술 스택**: 순수 HTML5, CSS3, Vanilla JavaScript (외부 프레임워크 미사용)
- **데이터 관리**: 서버 없이 브라우저의 `localStorage`를 활용하여 회원 정보, 소식(Stories), 사진 설정 등을 영구 저장함.

## 2. 주요 기능 및 설정
### 👤 회원 시스템 및 관리자 모드
- **통합 로그인**: 모든 사용자는 동일한 '로그인' 버튼을 통해 접속함.
- **숨겨진 관리자(Admin)**: 지정된 관리자 계정으로 로그인 시 관리자 전용 UI(대시보드, 수정 버튼 등)가 활성화됨.
- **동반자 라운지(My Page)**: 일반 회원이 로그인하면 전용 '디지털 동반자 카드'와 감사 메시지를 확인할 수 있음.

### 📰 소식 관리 (CRUD)
- 관리자는 최근 소식을 추가, 수정, 삭제할 수 있음.
- **이미지 업로드**: 경로 입력 방식과 더불어 컴퓨터의 파일을 직접 선택해 Base64로 저장하는 기능을 지원함.

## 3. 엔지니어링 표준
- **Surgical Edits**: 파일 전체를 덮어쓰기보다 `replace` 도구를 사용하여 필요한 부분만 정교하게 수정할 것.
- **LocalStorage 일관성**: 데이터 스키마(`theround_users`, `theround_stories`, `theround_logged_in_user` 등)를 엄격히 준수할 것.
- **UI/UX**: '더라운드'의 따뜻하고 포용적인 브랜드 이미지를 유지하기 위해 부드러운 애니메이션과 고급스러운 디자인 톤을 유지할 것.
